Debug Hotkeys!!
===============
Record any debug hotkeys you've created here,
so that everyone knows about them!

Camera
------
'M' - Hold to allow control of camera with
	  arrow keys and 'Q' & 'E' (Disabled for now because it breaks the game some how)
	  
'N' - Show buffer rectangle and circle around camera target

'O' - Changes all platform from dynamic to kinematic and back again

'T' - rotates all platforms

'Y' - stops rotating platforms

'P' - exit game
